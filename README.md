# CSharp Playground

A personal learning repository for exploring C# and .NET concepts hands-on, built alongside a Microservices with .NET 8 course.

Each folder under `src/` represents an isolated concept to be practiced with its necessary files: self-contained models, interfaces, and a `Runner.cs` that executes the example. 


## Structure
```
CSharpPlayground/
└── src/
    ├── Generics/        # Type variance: covariance, contravariance, invariance
    └── ...
```

## How to run

Open the solution in Visual Studio, set `CSharpPlayground` as the startup project, and press `F5`.
To switch between topics, update `Program.cs` to call the relevant `Runner.Run()`.

## Stack

- .NET 8
- C# 12
